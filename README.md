Walmart Review Analyzer

A complete **Transformer-based NLP web application** that analyzes customer reviews and provides real-time sentiment insights along with interactive visualizations.

---

## 🚀 Project Overview

The **AI Review Intelligence System** is designed to analyze customer feedback using a **pre-trained Transformer model (BERT-based)**. It predicts whether a review is **Positive or Negative** and provides additional insights through dashboards and word cloud visualization.

This project is built using **Streamlit** for an interactive and user-friendly interface.

---

## 🧠 Key Features

* 🔍 **Real-time Sentiment Analysis**

  * Predicts review sentiment instantly
  * Displays confidence score

* 📊 **Interactive Dashboard**

  * Sentiment distribution (Pie Chart)
  * Product-wise sentiment analysis (Bar Chart)
  * Key metrics display

* ☁️ **Word Cloud Visualization**

  * Shows most frequent words in reviews
  * Helps identify common themes

* ⚡ **Fast & Accurate Predictions**

  * Uses pre-trained Transformer (BERT)
  * No need for manual model training

* 🎨 **User-Friendly Interface**

  * Built with Streamlit
  * Clean and responsive layout

---

## 🏗️ Tech Stack

* **Programming Language:** Python

* **NLP Model:** HuggingFace Transformers (BERT-based)

## ⚙️ Installation Guide

### 1️⃣ Clone the Repository

```

cd ai-review-intelligence
```

### 2️⃣ Install Dependencies

